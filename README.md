# 我的Git練習(Command篇)

__一般的 Commit__
`git commit -m "說明文字"`

__沒東西也能 Commit__
`git commit --allow-empty -m '沒內容物的Commit' `

__不想給說明文字的 Commit__
`git commit -allow-empty-message -m ''`

__一個指令、兩個滿足 Commit & Add__
`git commit -a -m '說明文字'`

(-a 只對已經存在Repository的檔案有效，對Untracked file是無效的)

__檔案更名__
`git mv fileName`

__移除檔案__
`git rm fileName`
