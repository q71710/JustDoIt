# 我的Git練習 - Command篇

一般說明

`-m '文字'`

沒東西也能 Commit

`--allow-empty`

就是不想給說明

`--allow-empty-message`

附加Add功能

`-a -m '說明文字'`
> -a 只對已經存在Repository的檔案有效，對Untracked file是無效的

檔案更名

`mv fileName`

移除檔案

`rm fileName`
